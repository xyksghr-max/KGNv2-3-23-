## 1. 使用原则

- 本地默认工作区：`~/KGN-main`
- AutoDL 云端推荐工作区：`/root/autodl-tmp/KGN-main`
- 当前主工作流：本地负责代码修改、文档更新、轻量检查、commit 和 push；云端负责 pull 最新分支、GPU 训练、测试评估和保存实验结果。


